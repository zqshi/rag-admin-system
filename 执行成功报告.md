## 使用指南

### 上传文档：
1. 打开 http://localhost:8080
2. 点击"选择文件"
3. 上传PDF/TXT/MD文档

### 智能问答：
1. 在搜索框输入问题
2. 点击"搜索"按钮
3. 查看答案和相关片段

### API调用：
```bash
# 上传文档
curl -X POST "http://localhost:8000/api/upload" \
  -F "file=@document.pdf"

# 查询
curl -X POST "http://localhost:8000/api/query" \
  -H "Content-Type: application/json" \
  -d '{"query": "你的问题", "top_k": 5}'
```
